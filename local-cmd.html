# Indroduction

**【Local Cmd】**app, just like it's name, execute local machine's command tools. For example, ps、cat、df tools in linux system.

# Parameter

Os

Now only support unix or linux system.

Cmd

Local command tools.

Timeout

The maximum execution time of command tools.

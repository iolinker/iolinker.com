## 什么是 IOLinker？

IOLinker 是一款面向开发者的轻量级 AI 工作流引擎，融合了传统工作流的确定性与 AI 智能体的动态决策能力。安装包不到 100MB，运行内存不到 100MB，单文件部署，开箱即用。

## 核心特性

### ⚡ 极致轻量
- 安装包 < 100MB
- 运行内存 < 100MB
- 单文件部署，无需 Docker 和数据库
- 支持 Mac / Linux / Windows / Docker / 树莓派 / NAS

### 🤖 智能体工作流
融合传统工作流的确定性与 AI 智能体的动态决策，可视化编排 Agent 推理链路，兼顾稳定与智能。

### 🔗 多模型接入
兼容 OpenAI、Claude、GLM、Qwen 等主流模型服务，也支持接入本地模型，灵活切换不受限。

### 💬 多平台 Agent 部署
将 AI Agent 一键接入 Telegram、飞书、钉钉等平台，快速搭建具备 AI 能力的 Bot。

### 🧩 丰富的组件
内置触发器、逻辑组件、变量表达式、表单自动化、文件管理和数据分析等模块。

### 🧠 知识库 RAG
内置向量检索增强生成引擎，上传文档即刻构建专属知识库，让 AI Agent 拥有你的领域知识。

## 应用场景

- **多平台 Bot** - 部署 AI Agent 到 Telegram、飞书、钉钉等平台
- **AI Agent 工作流** - 可视化编排复杂的智能体推理链路
- **DevOps 自动化** - 编排运维工作流，自动巡检、分析、告警
- **表单与流程自动化** - 自动化处理表单数据、文件管理和数据分析

## 快速开始

1. **下载** - 选择你的平台，下载安装包
2. **连接模型** - 填入 API Key（OpenAI、Claude、GLM、Qwen 等）
3. **编排发布** - 在可视化画布上拖拽编排工作流，发布为 API 或部署到各平台
